レポート課題2 提出ファイル一覧
=====================================

このディレクトリには以下のファイルが含まれています：

【ソースファイル】
- report2_main.c : メインプログラム（エントリーポイント）
- shell.h        : ヘッダファイル（構造体・関数定義）
- shell.c        : 基本機能の実装（初期化、解析、ヒストリ等）
- builtins.c     : 内部コマンドの実装（cd, pushd, popd等）
- execute.c      : コマンド実行とパイプライン処理

【ビルド関連】
- Makefile       : コンパイル設定ファイル
- mysh           : コンパイル済み実行ファイル

【提出方法】
1. CLE提出用：
   - このディレクトリのファイルを tar.gz 形式でアーカイブ
   - コマンド: tar -czf <学籍番号>.tar.gz *
   - レポートPDFと一緒にCLEにアップロード

2. 演習室提出用：
   - mysh を以下のディレクトリにコピー
   - /home/exp/class/proc/$USER/submission/report2/mysh

【コンパイル方法】
$ make clean
$ make

【実行方法】
$ ./mysh
